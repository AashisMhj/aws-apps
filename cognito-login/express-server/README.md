# Cognito Login Express Server
The Backend Part of the Cognito Login

## Technologies
- express
- aws-sdk v3

## Prerequisite
- Cdk Deployed
- 

## Setup
```bash
## Install Packages
pnpm install
## copy cognito pool, client id and api url
## create .env file and fill all the values
## start server in watch mode
pnpm dev-watch
## build app 
pnpm build
```
