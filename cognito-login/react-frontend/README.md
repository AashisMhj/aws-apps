# Cognito Login: React App
The Frontend UI for 'Cognito Login' mini app.

Design from: https://github.com/BupendraDhyan/Login-Page-MUI
## Technologies
- React
- Vite
- Material UI

## Prerequisite
- Node Package Manger (pnpm preferred)

```bash
# install packages
pnpm install
## create .env file and copy from .env.example
## run app 
pnpm dev
```
